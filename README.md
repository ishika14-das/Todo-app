# To Do App

A simple task management application built using HTML, CSS and JavaScript.

## Features

✅ Add tasks  
✅ Delete tasks  
✅ Mark tasks as completed  
✅ LocalStorage support  
✅ Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
